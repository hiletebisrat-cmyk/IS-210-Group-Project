# Library Book Management System

A simple Java console application for managing books in a library.
By: James Maraden, Cesar Rosales Rodarte, Bisrat Hilete

## Features

- Add books
- Remove books
- Search books
- Display all books
- Check out books
- Return books
- Load books from a file

## Files

- `Main.java`
- `Book.java`
- `Library.java`
- `books.txt`

## Compile

```bash
javac *.java
```

## Run

```bash
java Main
```

## Sample File Format

```text
title,author,ISBN,publicationYear,status
```
